# Image Embedding Sample

Demonstrates image embedding generation using the CLIP ViT-Base model from the Model Garden.

## Prerequisites

- .NET 10.0 SDK
- Test image files named `cat.jpg`, `dog.jpg`, and `cat2.jpg` in the working directory

## Run

```bash
dotnet run
```

The model (~353 MB) will be downloaded automatically on first run and cached for subsequent runs.
